# Features

uSupport is split between two main backoffice experiences:

* The **Content** dashboard, where editors create and follow their own tickets
* The **uSupport** section, where support staff manage tickets, types, statuses, and settings

Core feature areas:

* [Tickets](/uSupport-documentation/docs/features/ticket) with searchable active and resolved lists
* Customizable [ticket types](/uSupport-documentation/docs/features/ticketType) with optional Umbraco property editors
* Customizable [ticket statuses](/uSupport-documentation/docs/features/ticketStatus) that define whether a ticket is active or resolved
* [Comments](/uSupport-documentation/docs/features/comment) for ticket follow-up
* [Settings](/uSupport-documentation/docs/features/settings) for notification behavior and email templates
